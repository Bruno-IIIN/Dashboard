# Comparador Inteligente de Planilhas

Aplicativo desktop moderno para comparação inteligente de planilhas Excel.

## Recursos

- Comparação automática
- Dashboard moderno
- Gráficos interativos
- Exportação Excel/PDF
- Tema Dark/Light
- Histórico de comparações
- Build portátil

## Tecnologias

- Python 3.11
- PySide6
- Pandas
- Plotly
- OpenPyXL
- PyInstaller

## Como executar

```bash
pip install -r requirements.txt
python main.py
